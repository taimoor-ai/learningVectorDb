# Introduction

Modern AI applications do not only generate text. They also need to search, understand, and retrieve information from custom data.

Vector databases solve this problem by storing information as mathematical representations called **embeddings**.

This repository explains:

- How text becomes vectors
- How vectors are stored
- How similarity search works
- How RAG systems are built
- How AI can use your own data
